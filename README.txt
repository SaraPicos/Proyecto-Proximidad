Proyecto sensor de proximidad 
Este proyecto ha sido desarrollado por los alumnos Borja Ferreiro, Alba Lama, Sara Picos y Lara Vila para la asignatura de sistemas de control en tiempo real.
Consiste en el control de un sensor de proximidad por código de colores: 
led rojo -> menos de 10 cm
led amarillo -> menos de 30 cm

led verde -> lejos

MONTAJE CIRCUITO
